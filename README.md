# RemoteESP32
用ESP32组装一个万能遥控器

## 注意：本项目含有大量AI生成内容，笔者只进行了基本可行性验证

### 功能
基于硬写和IRremoteESP8266的TV遥控与基于IRremoteESP8266的空调控制，允许自定义信号，带信号检测与解析，支持局域网连接与控制

### 使用
短按选中/进入选项，长按返回上一级，旋转旋钮上下选择，其余部分认识中文的话自己试就好

### TV遥控注意
TV遥控协议与编码复杂，内置预设只是象征意义，建议使用自定义模式学习信号
