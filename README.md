# RemoteESP32
用ESP32组装一个万能遥控器

### 连接
|硬件模块|模块引脚|接线目标|避坑指南 / 说明|
|--------|-----|-----|------------|
|屏幕 (OLED)|VSS|GND|VSS 代表负极/地线，接面包板 GND 轨。|
||VDD|3V3|VDD 代表正极/逻辑电源| 接面包板 3V3 轨。
||SCK|D22|硬件 I2C 时钟线 (Clock)。|
||SDA|D21|硬件 I2C 数据线 (Data)。|
|旋钮 (编码器)|5V|3V3|⚠️ 保命操作：无视它的丝印，必须接 3.3V！|
||GND|GND|接面包板 GND 轨。|
||S1|D32|旋转信号 1 (有的板子叫 A相/CLK)。|
||S2|D33|旋转信号 2 (有的板子叫 B相/DT)。|
||KEY|D25|旋钮按下时的按键信号。|
|发射|5V (或 VCC)|VIN (或 5V)|⚠️ 必须接 5V！榨干 USB 电流来驱动大功率 MOS 管发光。|
||GND|GND|接面包板 GND 轨。|
||IN (或 DAT/S)|D26|接收 ESP32 发出的高频闪烁指令。|
|接收 (VS1838B)|VCC (或 +)|3V3|接收模块常规供电，接面包板 3V3 轨。|
||GND (或 -)|GND|接面包板 GND 轨。|
||OUT (或 S/Y)|D27|用于读取旧遥控器的红外波形。|

### 功能
基于硬写和IRremoteESP8266的TV遥控与基于IRremoteESP8266的空调控制，允许自定义信号
